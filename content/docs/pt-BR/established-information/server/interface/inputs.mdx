O Hytale foi projetado para ser compatível com uma variedade de métodos de entrada para acomodar diferentes preferências de jogadores e plataformas.

## Sistema de interação

O Hytale conta com um sistema de interação poderoso que concede aos plugins a capacidade de personalizar a interpretação das atribuições de teclas e as ações que elas acionam. Esse sistema é orientado por dados, permitindo que os criadores de mods criem interações complexas por meio de ativos.

Os criadores de mods podem definir atribuições de teclas personalizadas para diversas ações, como usar habilidades ou interagir com objetos no mundo do jogo. O sistema de interação foi projetado para ser poderoso, embora exija dedicação para lidar com a complexidade dos ativos.

> Você pode usar as suas próprias interações para atribuir teclas para o que desejar. O sistema de interação é potente, mas ainda precisamos trabalhar em sua usabilidade, uma vez que os ativos podem se tornar bastante complexos e difíceis de gerenciar.
