# Contribuindo para o Hytale Modding

Agradecemos pela contribuição da comunidade para ajudar a melhorar e expandir a documentação de HytaleModding. Seja corrigindo erros de digitação, adicionando tutorias, ou providenciando exemplos de código,  a sua contribuição  é valiosa.

## Como contribuir

### Relatando problemas

Se você encontrou um problema com a documentação, reporte-o abrindo uma nova issue em nosso [repositório do GitHub](https://github.com/HytaleModding/site/issues).

Se você pode trabalhar nesse problema você mesmo, pode pedir para ser atribuído a ele e enviar um PR.

### Enviando pull request

1. Faça um fork no repositório do GitHub
2. Clone seu fork para sua maquina local
3. Crie uma branch para as suas alterações

```bash
git checkout -b my-feature-branch
```

4. Faça as alterações e envie-as com mensagens limpas, objetivas e claras. Seguimos [commits convencionais](https://www.conventionalcommits.org/en/v1.0.0/), seu PR não será aceito se suas mensagens de commit não seguirem esta formatação.

```bash
git commit -m "Adicionar recurso X à documentação"
```

5. Envie suas alterações para o seu fork.

```bash
git push origin my-feature-branch
```

6. Abra um Pull Request no repositório original, descrevendo suas alterações e por que elas devem ser mescladas.
