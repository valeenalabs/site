## Por que aprender a programar? Temos a programação visual!

Sim, temos, e honestamente você deveria tentar-los. Visual scripting é uma ótima adição para o Hytale e sabemos que ela será poderosa, mas nada bate a capacidade de ler e escrever códigos Java, com a habilidade de construir mods e plugins ainda mais poderosos.

<Callout type="info" title="Programação Visual vs. Codificação">
  A programação visual é ótima para iniciantes e tarefas simples, mas a codificação permite soluções mais complexas e eficientes. Aprender a programar proporciona diversas possibilidades para criação de mods no Hytale.

  Teremos guias sobre programação visual disponíveis a partir do dia 1 do lançamento de Hytale.
</Callout>

## Todos os criadores de mods para o Hytale precisam aprender a programar?

Ninguém deveria sentir que PRECISA aprender a programar para fazer coisas no Hytale. Simon e sua equipe já provaram isso várias vezes com o avanço dos editores gráficos e do sistema de programação visual. Você pode alterar o comportamento das criaturas, a geração do mundo e muito mais, apenas com essas ferramentas.

No entanto, se o seu objetivo é criar projetos mais complexos (p. ex.: de utilitários a softwares de servidor personalizados do zero) que podem ser usados por grandes redes e influenciadores, aprender a programar se torna indispensável. E é aí que este guia entra...

## OK! Tenho interesse. O que vem a seguir?

Ao longo das próximas aulas, você aprenderá o básico de programação, como ler a documentação e a como encontrar recursos que lhe ajudarão a aprender mais por conta própria.

Os guias adotarão uma perspectiva de "visão panorâmica" e fornecerão recursos externos onde você poderá realizar a programação. Isso é feito porque, sinceramente, já existem recursos suficientes para aprender tudo isso, o difícil é encontrar o material certo no Google. Aprender a programar sem depender de instruções passo a passo é o objetivo. A comunidade de modding do Hytale estará sempre ali para ajudar, mas para se tornar um desenvolvedor realmente completo você vai precisar absorver muita coisa simplesmente lendo materiais e recursos por conta própria.

<Callout type="info" title="Procurando recursos">
  Criamos uma pequena série sobre os conceitos básicos do Java para que você se familiarize com a linguagem. No entanto, após essa etapa, será necessário buscar recursos adicionais para aprofundar seus conhecimentos. De início, recomendamos sites como [w3schools](https://www.w3schools.com/java/) e [GeeksforGeeks](https://www.geeksforgeeks.org/java/) para quem está começando.
</Callout>

Nosso objetivo é ensinar você a como criar mods para o Hytale, e esse sempre será nosso objetivo. Por essa razão, não conseguiremos abordar tudo de Java ou de programação em geral. Em vez disso, buscamos entregar os meios para que você aprenda de forma independente, garantindo seu crescimento contínuo como desenvolvedor, mesmo após terminar nossos guias.

## Alguma consideração final?

Lembre-se, aprender a programar é difícil. Você vai tropeçar e cair, mas conseguirá se levantar de novo. Mantenha a resiliência, acredite que consegue criar coisas incríveis e eu prometo que o sucesso virá. Boa programação!
