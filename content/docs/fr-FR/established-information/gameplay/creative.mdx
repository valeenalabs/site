# Mode créatif

Le mode créatif d'Hytale est un mode bac à sable qui permet aux joueurs de modifier le monde librement.

## Outils d'édition du monde

Le mode créatif comprend divers outils pour aider les joueurs à modifier le monde. Cela inclut la possibilité d'annuler des modifications.

Notez que les sections ci-dessous n'étaient pas nommées dans la vidéo de démonstration des outils.

---

### Pointeur laser

Affiche un rayon partant de la caméra du joueur dans la direction que regarde le joueur.

#### Paramètres

- Durée (Slider entier) : Description inconnue en décembre 2025.
- Couleur du laser (Couleur en code hexadécimal) : Description inconnue en décembre 2025.

### Outil pinceau de sculpture

Permet à l'utilisateur de faire monter ou s'abaisser le terrain pour sculpter et mélanger les formes de manière plus précise. Supporte la plupart des fonctionnalités du pinceau.

### Outil pinceau de ligne

Inconnu en décembre 2025.

### Outil de sélection

Permet à l'utilisateur de sélectionner et modifier des portions du monde en transformant ces sélections de blocs ou un préfabriqué de manière à changer leur position, leur rotation, ou d'en supprimer certaines parties.

#### Paramètres

- Mode (Liste déroulante : Poser, Mur, Remplir, Remplacer) : description inconnue en décembre 2025.
- Matériel (Curseur entier) : description inconnue en décembre 2025.
- Poids (Slider entier) : description inconnue en décembre 2025.
- Pitch (Entier) : description inconnue en décembre 2025.

### Règle

Permet à l'utilisateur de mesurer la distance entre plusieurs points dans l'espace 3D, avec la possibilité d'avoir plusieurs lignes de mesure et de modifier les points de mesure.

### Outil pinceau de claque

Inconnu en décembre 2025.

### Outil coller

Permet à l'utilisateur de coller une sélection préalablement copiée de blocs ou d'un préfabriqué à l'endroit où le joueur regarde. Le collage est relatif au point d'ancrage de la sélection. Si la sélection est un préfabriqué, le point d'ancrage peut être défini dans l'éditeur de préfabriqués. Si elle est inconnue, l'ancre de la sélection peut être définie autrement.

### Outil pinceau d'inondation

Inconnu en décembre 2025.

### Outil pinceau

Les pinceaux sont utilisés pour faire des modifications importantes au monde, même à grande distance.

#### Capacités générales

- Placer des formes et des préfabriqués
- Remplacer des blocs
- Créer des verrouillages d'axe et des masques

#### Paramètres

- Matériel
- Largeur (Slider entier) : description inconnue en décembre 2025.
- Hauteur (Slider entier) : description inconnue en décembre 2025.
- Forme (Liste déroulante : Cube, Sphère, Cylindre, Cône, Cône renversé, Pyramide, Pyramide renversée, Grille Cubique) : description inconnue en décembre 2025.
- Origine (Liste déroulante : Centre, Bas, Haut) : description inconnue en décembre 2025.
- Axes miroirs (Liste déroulante : Aucun, Autres options inconnues) : description inconnue en décembre 2025.
- Épaisseur du mur (Slider entier) : description inconnue en décembre 2025.
- Densité (Slider entier) : description inconnue en décembre 2025.

#### Prédéfinis :

- Bloc de roche : crée un bloc de roche à l'endroit où le joueur regarde.
- Chemin d'herbe : transforme le terrain en un chemin d'herbe à l'endroit où le joueur regarde.
- Gravats : génère des gravats à l'endroit où le joueur regarde.
- Tentacule : à déterminer.
- Fissures de lave : transforme le terrain en fissures de lave à l'endroit ou le joueur regarde.
- Sources chaudes : à déterminer.
- Ruines : génère une ruine à l'endroit ou le joueur regarde.
- Toits en ruine : génère une ruine avec un toit à l'endroit ou le joueur regarde.
- Herbe : à déterminer.
- Montagne : Place des morceaux de terrain consécutifs pour former une montagne à l'endroit ou le joueur regarde.
- Ruines : à déterminer.

### Machinima

Le machinima est un éditeur de séquence intégré, permettant à l'utilisateur de créer et modifier des séquences d'actions pour les rejouer en jeu. Cela comprend un éditeur interactif d'images clés.

La vitesse de lecture peut être ajustée par des commandes.

L'utilisateur peut ajouter des acteurs pour la séquence à contrôler.

- Référence
- Caméra
- Entité
- Objet
- Joueur

## Commandes

Les commandes sont entrées dans la console de discussion du monde et débutent avec une barre oblique `/`.

- `set <block>` : permet à l'utilisateur de transformer les blocs d'une sélection en un bloc spécifié.
- `inventory clear` : permet à l'utilisateur de vider leur inventaire.
