Bienvenue dans votre voyage au cœur de la programmation Java ! Ce guide est conçu pour vous introduire aux bases de Java.

## Qu-est-ce que Java ?

Java est un puissant langage de programmation orienté objet créé en 1995. C'est l'un des langages de programmation les plus populaires au monde et est utilisé pour tout, des applications mobiles aux logiciels d'entreprise. Plus important pour nous, c'est le langage qu'Hytale utilise pour ses serveurs et son API de modding.

## Ce que vous allez apprendre

Ce guide est structuré pour vous accompagner étape par étape à travers :

1. Syntaxe et bases de Java
2. Structures de contrôle
3. Programmation orientée objet

Nous nous concentrerons ensuite sur les concepts spécifiques à Hytale. (à déterminer)

## Configurer votre environnement

Avant de plonger dans la programmation, vous aurez besoin de quelques éléments essentiels, vous devez installer :

### Le Java Development Kit (JDK)

Téléchargez et installez le JDK 25 ou plus depuis le [site officiel d'Oracle](https://www.oracle.com/java/technologies/javase-jdk25-downloads.html) ou utilisez une alternative open-source comme [AdoptOpenJDK](https://adoptopenjdk.net/).

### Environnement de développement intégré (IDE)

Un IDE est une application logicielle qui fournit des outils complets aux développeurs pour le développement logiciel. Les choix populaires pour Java sont :

* [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* [Eclipse](https://www.eclipse.org/)

## Comment utiliser ce guide

1. Lisez chaque article dans l'ordre - les concepts se superposent
2. Tapez manuellement les exemples - ne faites pas de copier-coller ; taper vous aide à apprendre
3. Expérimentez - modifiez les exemples pour voir ce qu'il se passe
4. Pratiquez - chaque article comprend des exercices
5. Soyez patients - la programmation est une compétence qui prend du temps à se développer

## La mentalité du modding

Le modding est une résolution de problème créative. Vous allez :

* Ajouter de nouveaux blocs, objets et créatures dans Hytale
* Changer des mécaniques de jeu pour créer des expériences uniques
* Construire des systèmes qui interagissent avec le monde du jeu
* Partager vos créations avec la communauté
