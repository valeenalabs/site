Sāksim ar 3D Modelēšanas pamatiem.

## Download & Installing Blockbench

Hytale uses Blockbench for all 3D modeling tasks. You can download it from the official website: [Blockbench](https://www.blockbench.net/).

Pēc lejuplādes seko instalācijas instrukcijām attiecīgajai operētājsistēmai.

<Callout>
  Tu vari arī izmantot BlockBench [web versiju](https://web.blockbench.net/), ja nevēlies to instalēt, bet tas nav ieteicams ar lieliem/sarežģītiem modeļiem.
</Callout>

<Callout type="warning">
  If you are using **Linux**, pick the package supported by your current Linux Distribution. If you are unsure, you can download the Blockbench AppImage (Recommended), or the Flatpak package, both are universal installers and works on all Linux distributions.
</Callout>

Once you finish downloading and installing the package, you will see the following menu on first launch. Which indicates you've successfully installed Blockbench.

![Blockbench HomePage](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_homepage.png)

Now, we need to install the Hytale Blockbench plugins, for that, follow these steps:

1. Click on File on the top left corner.
2. Select "Plugins"
3. You will see a Hytale Models plugin, if you don't, search for it and select it.
   ![Blockbench Plugins Overview](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_plugins_overview.png)
4. Click on **Install**

You're done! Now let's proceed with creating a model. If you're making a character model, like a NPC, animal etc. you should use the **Hytale Character** Model type. For this guide, we will focus on creating the Hytale Prop, but it is pretty similar. Click on the **Hytale Prop** model Type.

![Blockbench Hytale Character](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_hytale_character.png)

Now you'll be prompted to enter a file name, enter the name of your models.

<Callout type="warning">
  Do not add any special characters, spaces, symbols or non-latin characters (except for underscores).

  ✅ my_model

  ❌ مy m♥פel$
</Callout>

![Blockbench Hytale Character](https://cdn.internal.hytalemodding.dev/assets/guides/blockbench_hytale_character_new_project.png)

That's it! You now have a new Blockbench Hytale Character Model! In our next guides, we will learn how to actually create this model.
