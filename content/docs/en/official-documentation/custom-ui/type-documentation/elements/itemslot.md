# ItemSlot

**Accepts child elements:** No

A self-contained item slot element for Custom UI that renders like native inventory slots. Displays item icon, quantity, quality background, and durability bar. This is a display-only element - use inside an ItemSlotButton for click handling. Tooltips are handled by the parent ItemSlotButton which detects ItemSlot children. Can be used in two modes: 1. Config-based: Set ItemId and Quantity (for shops, previews where items don't exist yet) 2. Instance-based: Call SetItemStack() with a ClientItemStack (for actual inventory items) Example usage in .ui file: ItemSlotButton #TradeButton { ItemSlot #OutputSlot { Anchor: (Width: 48, Height: 48); } } Then from server: set("#OutputSlot.ItemId", "Food\_Bread"); set("#OutputSlot.Quantity", 5);
